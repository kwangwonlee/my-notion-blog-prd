# 월별 독서 타임라인 (Monthly Reading Timeline) - PRD

## 📋 프로젝트 개요

### 프로젝트 정보
- **프로젝트명**: 월별 독서 타임라인 (Monthly Reading Timeline)
- **버전**: 1.0.0 (MVP)
- **작성일**: 2026-01-19

### 목적
Notion을 CMS로 활용한 개인 독서 기록 및 리뷰 공유 웹사이트를 구축하여, 시간순으로 정리된 독서 이력을 시각적으로 제공하고 독서 경험을 다른 사용자와 공유합니다.

### CMS 선택 이유
- **Notion API 활용**: 비개발자도 손쉽게 콘텐츠를 관리하고 업데이트할 수 있음
- **무료 사용 가능**: 개인 프로젝트에 적합한 비용 효율성
- **구조화된 데이터**: 데이터베이스 형태로 체계적인 콘텐츠 관리
- **즉시 반영**: Notion에서 수정한 내용이 웹사이트에 실시간으로 반영

## 🎯 주요 기능

### 1. 월별 독서 타임라인 표시
- 읽은 책을 월별로 그룹핑하여 타임라인 형태로 표시
- 최신 월부터 역순으로 정렬
- 각 월별 섹션에 해당 월의 독서 권수 표시
- 스크롤 기반의 직관적인 네비게이션

### 2. 책 정보 카드
각 책 카드에는 다음 정보를 표시:
- **표지 이미지**: 책의 커버 이미지
- **제목**: 책 이름
- **저자**: 저자명
- **읽은 날짜**: 완독 날짜
- **별점**: 5점 만점의 별점 평가
- **장르 태그**: 책의 장르 정보
- **한줄평**: 짧은 요약 (옵션)

### 3. 장르별 필터링 기능
- 상단에 장르 필터 버튼 그룹 제공
- 다중 선택 가능한 필터
- 필터 적용 시 해당 장르의 책만 표시
- "전체 보기" 옵션으로 필터 초기화

### 4. 통계 요약
- **총 독서량**: 전체 기간 동안 읽은 책의 총 권수
- **이번 달 독서량**: 현재 월에 읽은 책의 권수
- **장르별 분포**: 각 장르별 독서 비율 (선택 사항)

### 5. 책 상세 정보
- 책 카드 클릭 시 상세 정보 표시 (모달 또는 페이지)
- 모든 기본 정보 + 상세 리뷰 전문
- 소셜 공유 버튼 (향후 확장)

## 🗄️ Notion 데이터베이스 구조

### 데이터베이스명: `📚 독서 기록`

| 속성명 | 타입 | 설명 | 필수 여부 |
|--------|------|------|-----------|
| **책 제목** | Title | 책 이름 | 필수 |
| **저자** | Text | 저자명 | 필수 |
| **표지 이미지** | Files & media | 책 표지 이미지 URL | 필수 |
| **읽은 날짜** | Date | 완독한 날짜 | 필수 |
| **별점** | Select | ⭐⭐⭐⭐⭐, ⭐⭐⭐⭐, ⭐⭐⭐, ⭐⭐, ⭐ 중 선택 | 필수 |
| **장르** | Multi-select | 소설, 에세이, 자기계발, 기술, 역사, 철학, 과학 등 | 필수 |
| **한줄평** | Text | 짧은 요약 (최대 100자) | 선택 |
| **상세 리뷰** | Rich text | 자세한 독서 후기 | 선택 |
| **상태** | Select | 읽는 중, 완독, 중단 (필터링용) | 선택 |
| **ISBN** | Text | 책 고유 번호 (향후 확장용) | 선택 |

### 샘플 데이터 예시
```
책 제목: "클린 코드"
저자: "로버트 C. 마틴"
표지 이미지: [이미지 파일]
읽은 날짜: 2026-01-15
별점: ⭐⭐⭐⭐⭐
장르: 기술, 자기계발
한줄평: "개발자라면 반드시 읽어야 할 필독서"
상세 리뷰: "코드의 품질을 높이는 실용적인 방법들을 배울 수 있었습니다..."
상태: 완독
```
